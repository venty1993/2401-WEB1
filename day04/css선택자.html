1. id 선택자 (#) : 우선순위 높음! - 한 문서에 딱 하나만 존재할수 있는 선택자 - 한 태그에 하나까지만 넣을수 있다 2. class 선택자 (.) - 한 문서에 여러개 존재할수 있는 선택자 - 한 태그가 여러 클래스를 동시에 가질 수 있다 - 여러 태그가 하나의 클래스를 같이 가질 수 있다. 3. 태그 선택자 - HTML 태그를 활용하는 선택자 - 디자인을 넣기위해 사용한다기보단 빼기위해 사용할때가 많다 위 선택자들을 조합해서 사용 가능하다. p.영업팀#박영희 { css속성 } css는 우선순위에 따라서 속성이 무시 되는 방식을 채택하고있다. 우선순위는 기본적으로 아래있는 작성이 높게 책정이 되고 선택자의 명확도, id인지 class인지 여부등에 의해 결정이 된다. 이 부분이 처음 css를 배울때 진입장벽이 되긴 하지만 숙달된 이후 잘 사용한다면 굉장히 효율적인 방식이기도 하다.
